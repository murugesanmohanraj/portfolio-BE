# Snack Quality Management System

## Overview

The Snack Quality Management System is a web application developed at Kitkat Software Technologies to help food manufacturing businesses manage product quality, inspection processes, production records, and quality control activities.

Mohanraj Murugesan contributed as a Software Engineer, developing web applications, backend services, and REST APIs to support quality assurance workflows and improve operational efficiency.
