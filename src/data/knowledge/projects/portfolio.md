# Personal Portfolio Website

## Overview

The Personal Portfolio Website is a modern web application built and maintained by Mohanraj Murugesan to showcase his professional experience, technical skills, projects, achievements, and career journey. The website serves as a central hub for recruiters, hiring managers, and developers to learn about his work and explore his projects.

The portfolio is continuously updated as new technologies, projects, and professional achievements are added.
