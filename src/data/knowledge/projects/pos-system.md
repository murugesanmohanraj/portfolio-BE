# Point of Sale System

## Overview

The Point of Sale System is a retail application built to support billing, inventory management, payment processing, and multi-store operations.

Mohanraj contributed to this solution while working at Millennium IMS Pvt Ltd.

---

## Purpose

The system helps retail businesses manage sales, track inventory, process payments, and streamline day-to-day operations.

---

## Technologies

- JavaScript
- Node.js
- Express.js
- MongoDB
- REST APIs
