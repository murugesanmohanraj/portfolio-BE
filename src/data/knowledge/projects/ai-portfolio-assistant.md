# AI Portfolio Assistant

## Overview

The AI Portfolio Assistant is an AI-powered chatbot integrated into Mohanraj Murugesan's personal portfolio website. It enables recruiters, hiring managers, and visitors to ask natural language questions about his experience, projects, technical skills, achievements, and career journey.

The assistant is designed to provide accurate, context-aware answers based only on Mohanraj's portfolio data instead of generating generic responses.

- Professional profile
- Work experience
- Projects
- Technical skills
- Achievements

- Interview knowledge

Visitors can ask questions naturally instead of navigating through multiple pages.

## Interview Talking Points

### Why did you build the AI Portfolio Assistant?

I wanted recruiters to interact with my portfolio conversationally instead of reading static pages. It also gave me an opportunity to learn how LLMs, prompt engineering, and knowledge retrieval work in real-world applications.

### What technical challenges did you solve?

One of the biggest challenges was ensuring the assistant answered only from my portfolio instead of inventing information. I addressed this by designing a structured knowledge base, writing clear system prompts, and planning a Retrieval-Augmented Generation architecture.

### What did you learn?

This project significantly improved my understanding of LLM integration, prompt engineering, backend AI APIs, knowledge organization, system design, and building production-ready AI applications.

### What are your future plans for the project?

I plan to implement Retrieval-Augmented Generation (RAG), semantic search using embeddings, vector databases, streaming responses, conversation memory, and support for local open-source language models to improve accuracy while reducing API costs.
