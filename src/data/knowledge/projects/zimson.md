# Zimson Retail Management System

## Overview

The Zimson Retail Management System is an enterprise retail application developed at Millennium IMS Pvt Ltd for Zimson, a leading watch retailer. The application supports daily retail operations, including product management, billing, inventory tracking, customer management, and sales workflows across multiple store locations.

Mohanraj Murugesan contributed as a Full Stack Software Engineer, developing customer-facing interfaces, backend services, and REST APIs while enhancing existing production features.
