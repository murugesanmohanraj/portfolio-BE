# Point of Sale (POS) System

## Overview

The Point of Sale (POS) System is a retail management application developed at Millennium IMS Pvt Ltd. It helps businesses manage billing, payments, inventory, products, customers, and multiple store locations through a single platform.

Mohanraj Murugesan contributed as a Full Stack Software Engineer, developing both frontend and backend features for the application.
