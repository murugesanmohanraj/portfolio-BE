# Dope Music

## Overview

Dope Music is a full-stack music streaming web application built by Mohanraj Murugesan using the MERN stack. The project was created to gain practical experience in building scalable web applications with authentication, music streaming, playlist management, and modern React architecture.

The application provides users with an interactive platform to discover music, create playlists, manage favorites, and stream audio through a responsive interface.
